Examples
---
### Version **0.0.1 (alpha)**
Code:
```ruby
output: "Hello, World!"

x = 2
greeting = "Welcome, User!"

output: 2 + x * 2
output: greeting
``` 

Result:
```ruby
Hello, World!
6
Welcome, User!
```

Installation
---

### Dependencies
> You need to have only "**Git**" and "**Python**".

> **Python** version is **3.14** *recommended*.

### Windows
```bash
git clone https://github.com/d1nozavr/DragonTree.git
cd DragonTree
python interpreter.py
```

### Linux/MacOS
```bash
git clone https://github.com/d1nozavr/DragonTree.git
cd DragonTree
python3 interpreter.py
```
