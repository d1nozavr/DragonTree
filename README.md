Installation
---

### Dependencies
> You need to have only "**Python**" and "**git**".

> **Python** version is **3.14** *recommended*.

### Windows
```
git clone https://github.com/d1nozavr/DragonTree.git
cd DragonTree
python interpreter.py
```

### Linux/MacOS
```
git clone https://github.com/d1nozavr/DragonTree.git
cd DragonTree
python3 interpreter.py
```
