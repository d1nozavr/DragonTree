# DragonTree
DragonTree - programming language.
Now, there are 'Token' & 'Lexer', it is not final version of '0.0.1 (alpha)'

## Installation

### Windows
```
git clone https://github.com/d1nozavr/DragonTree.git

cd DragonTree

python interpreter.py
```

### Linux
```
git clone https://github.com/d1nozavr/DragonTree.git

cd DragonTree

python3 interpreter.py
```
